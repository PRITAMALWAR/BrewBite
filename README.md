# ☕ BrewBite Café – Fast Food Website

A modern, responsive fast food café website built using **Next.js, React, and Tailwind CSS**.  
Designed for local businesses serving tea, coffee, sandwiches, patties, snacks, and quick bites.

---

## 🚀 Project Overview

**BrewBite Café** is a conversion-focused website designed to attract local customers and increase orders.  
It features a clean UI, mobile-first design, and essential business integrations like WhatsApp and Google Maps.

---

## 🎯 Features

- ⚡ Modern & responsive design
- 📱 Mobile-first layout
- 🎨 Warm café-style UI (brown, cream, orange theme)
- 🧭 Sticky navigation bar
- ✨ Smooth scrolling & hover animations
- 📍 Google Maps integration
- 📞 Click-to-call functionality
- 💬 WhatsApp chat button (floating)
- 🔍 SEO-friendly structure

---

## 🧱 Tech Stack

- **Frontend:** Next.js, React
- **Styling:** Tailwind CSS
- **Icons:** React Icons / Heroicons
- **Deployment:** Vercel (recommended)
